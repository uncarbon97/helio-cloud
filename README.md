# helio-cloud

## 项目介绍

`helio-cloud`基于 Spring Boot 2.6.x + Spring Cloud Alibaba 2021 + Dubbo，是一款预置SaaS、RBAC能力的微服务项目脚手架，助力开发者快速上手企业级微服务开发

JDK compatibility: 1.8 - 15

[官方文档](https://helio.uncarbon.cc/)


需要先安装 `MySQL`、`Redis` 等必需中间件

基础支撑构件 [helio-starters](https://github.com/uncarbon97/helio-starters) 已推送至Maven中央仓库，加载时会自动拉取
